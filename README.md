# 短連結網址


## 介紹

將網站的長網址，縮減為短網址

# 開始使用

1.請先確認有安裝 node.js 與 npm

2.將專案 clone 到本地

3.在本地開啟之後，透過終端機進入資料夾，輸入：

 `npm install`

4.安裝完畢後，繼續輸入：
 
 `npm run start`

5.若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址

 `Listening on http://localhost:3000`
 
6.若欲暫停使用

 `ctrl + c`
 
 # 開發工具
 
 * Node.js 14.16.0
 * Express 4.18.2
 * Express-Handlebars 4.0.2
 * dotenv 16.0.3
 * MongoDB
 * mongoose 5.9.7
